# Sweet Home

> Sistema embarcado para automação residencial utilizando a placa esp32.

## Estrutura

> Baseada na estrutura de organização da plataforma [PlatformIO](https://platformio.org)

## Linguagem

- C++

## Dependências

- Arduino
- WiFi
- WiFiUdp
- ESPmDNS
- ArduinoOTA
- aREST
- esp_system
- vector
- iostream
- functional
- map
- string
